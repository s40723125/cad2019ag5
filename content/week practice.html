week practice

week10-11

  1. 以分組協同方式整理教科書內容:

2015- Space Modeling with SolidWorks and NX
https://link.springer.com/book/10.1007/978-3-319-03862-9

各組員負責將課程重點與零組件繪圖練習, 加上解說與操作過程影片, 放入分組倉儲中.

2008- The Engineering Design Revolution
The Engineering Design Revolution (a.k.a. cad_history.pdf)

各組員負責編寫 Solidworks, NX, Inventor 與 Creo (Pro/E) 的開發緣起, 套件安裝與設定流程, 以及利用各套件執行零組件繪圖的相關差異與特點分析, 分別放入分組倉儲中.

  1. Github 分組倉儲如何 fork, 如何建立正向 pull requests, 如何反向 pull requests, 組長如何處理各組員的 pull requests 內容, 如何處理 content.htm, reveal 簡報以及 pelican 網誌 markdown 中的改版衝突. (含 GitExtensions 導入)

  2. NX12 零組件繪圖與 NXOpen Python 檔案的建立與應用. (http://mde.tw/cad2019/content/Task2.html)

  3. 各組員分別執行 V-rep 與 Webots Tutorial, 分別利用 Solvespace 與 NX 建立零組件轉入, 比較兩者差異 (http://mde.tw/cad2019/content/Task3.html)

  4. 分組執行 http://mde.tw/cad2019/content/Project.html 期末專案

week12

 NX 零組件繪圖

請根據 http://mde.tw/cad2019/downloads/NX-12-for-Engineering-Design.pdf 的導引, 透過分組協同完成各章節內容閱讀與操作練習, 並將練習影片加上說明字幕, 以校名+系名+課程名稱+分組任務名稱+學號作為標題, 上傳 Youtube 後, 分別嵌入個人與分組網頁中.

使用 NX3 者可以參考 http://mde.tw/cad2019/downloads/UG-NX3-Tutorial.pdf

目的在能使用 NX 建立 http://mde.tw/cad2019/content/Task2.html 完成越野車零組件的設計與繪圖, 並利用 CAE 模組進行零件的應力分析.

Webots 與 V-rep Tutorial

請各組員在第十三週之前完成:

https://cyberbotics.com/doc/guide/tutorial-1-your-first-simulation-in-webots

與

http://www.coppeliarobotics.com/helpFiles/en/bubbleRobTutorial.htm

並將操作過程製作成帶字幕影片, 上傳 Youtube 後, 分別嵌入個人與分組網站, 且將心得與兩者的比較內容放在個人與分組網誌中.

week13

每週將分別就課程任務執行結果與心得, 在個人與分組 Reveal 中進行各週簡報, 以作為期末自評與互評成績評定之參考.
